Контроллер оборотов мини сверлильного станка. Это модификация проекта https://alexgyver.ru/drill-pid/. Оригинальны код проекта находиться тут : https://github.com/AlexGyver/DrillPID
Изменения:
1. Можно выбрать управление энкодер или кнопки. Устанавливается в файле config.h
2. Нет рывков при изменении оборотов, обороты меняются плавно.
3. При включении питания состояние «Стоп»
4. При заклинивании переход в состояние «Стоп».
