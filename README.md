Контроллер оборотов мотора мини сверлильного станка. Это модификация проекта https://alexgyver.ru/drill-pid/ . Заменен энкодер на три кнопки, при включении станка состояние «Стоп», при заклинивании также переход в состояние «Стоп».
Оригинальны код проекта находиться тут : https://github.com/AlexGyver/DrillPID
